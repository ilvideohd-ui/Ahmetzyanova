Random Password Generator (Генератор случайных паролей)

Создать GUI-приложение «Random Password Generator» с возможностью настройки параметров пароля, сохранением истории и Git.

Пошаговая инструкция:

1. Создайте элементы интерфейса:

○ Ползунок длины пароля

○ Чекбоксы для выбора символов (цифры, буквы, спецсимволы)

○ Кнопка генерации

○ Таблица истории

2. Используйте библиотеку random для генерации.

3. Добавьте сохранение истории в JSON и загрузку обратно.

4. Проверяйте корректность ввода:

○ Минимальная/максимальная длина

5. Используйте Git:

○ Настройте репозиторий.

○ Выложите проект.

6. Напишите README (в формате .txt, .md), в котором укажите:

○ Имя и фамилию автора

○ Описание

○ Примеры использования (возможные тесты — по желанию)
